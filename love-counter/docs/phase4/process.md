- 同じLIFF ID、同じ公式アカウント
- メッセージ配列だけ変更
- URLを分ける（例：`/love-counter/` と `/love-counter-special/`）

## 🔧 技術的な実装

### リッチメニューの設定

**方法1: LINE Official Account Managerで設定**
- 視覚的に設定できる
- 簡単だが柔軟性は低い

**方法2: Messaging APIで設定**
- コードで管理できる
- 複数パターンを簡単に切り替え可能

### 自動応答の設定

**キーワード応答：**
```javascript
// api/webhook.js
if (message.text === '占い') {
  await client.replyMessage(replyToken, {
    type: 'text',
    text: 'LIFF URLを送信',
  });
}
```

### ステップ配信

**シナリオ例：**
1. 友達追加 → ウェルカムメッセージ
2. 3日後 → 使い方ガイド
3. 1週間後 → おすすめ機能紹介
