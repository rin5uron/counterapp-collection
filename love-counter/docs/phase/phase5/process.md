## 🚀 STEP1: リッチメニュー実装（12月中）

### 目標
**公式LINEっぽくする！**

リッチメニューを実装して、見た目と機能を一気に向上させる。

---

### 📋 実装内容

#### 1. アプリを開く
- **アクション**: URLを開く
- **URL**: `https://love-counter-theta.vercel.app/`
- **既にある**: ✅

#### 2. クーポンを開く
- **アクション**: クーポンを表示
- **内容**: 友達紹介クーポン（例：「友達紹介で○○プレゼント」）
- **作成時間**: 5分

#### 3. このアプリでできること
- **アクション**: テキストメッセージを送信
- **内容**:
  ```
  💕 このアプリでできること

  ✨ ランダムメッセージ
  りんごボタンを押すと、50種類のメッセージがランダムで表示されるよ！

  📸 画像送信
  22回目には返信フォームが表示されて、テキストや画像を送れるよ！

  💌 1対1のコミュニケーション
  ここはあなただけの特別な場所。一斉配信はしないから安心してね。

  使ってみたい方は「ねえねえ」と送ってね！
  ```
- **作成時間**: 3分

### 🎨 リッチメニューのデザイン

#### レイアウト（3分割）
```
┌─────────┬─────────┬─────────┐
│         │         │         │
│ アプリ  │クーポン │できること│

│         │         │         │
└─────────┴─────────┴─────────┘
```

#### 画像サイズ
- 推奨: 2500x843px（横長3分割）
- または: 1200x405px（簡易版）

#### 画像作成
- Canvaで簡単に作成可能
- テンプレート使用で5分

---

### ⚡ 実装手順（所要時間: 30分）

#### Step 1: 紹介文を作成（3分）
- 上記のテキストをコピー
- 必要に応じて調整

#### Step 2: クーポンを作成（5分）
1. LINE Official Account Managerにログイン
2. 「クーポン」→「作成」
3. 内容を入力（例：「友達紹介で次回使えるスペシャルクーポン」）
4. 保存

#### Step 3: リッチメニュー画像を作成（10分）
1. Canvaで2500x843pxのデザインを作成
2. 3分割のレイアウト
3. 各エリアにアイコンとテキスト
4. ダウンロード

#### Step 4: リッチメニューを設定（12分）
1. LINE Official Account Managerにログイン
2. 「リッチメニュー」→「作成」
3. 画像をアップロード
4. タップ領域を設定（3分割）
5. アクションを設定：
   - 左: URLを開く → `https://love-counter-theta.vercel.app/`
   - 中央: クーポンを表示 → 作成したクーポンを選択
   - 右: テキストを送信 → 紹介文を入力
6. 「公開」をクリック

## 🔧 技術的な実装

### リッチメニューの設定

#### 方法1: LINE Official Account Managerで設定
- **メリット**: 視覚的に設定できる、簡単
- **デメリット**: 柔軟性は低い

**手順：**
1. LINE Official Account Managerにログイン
2. 「リッチメニュー」→「作成」
3. 画像をアップロード（1200x810px）
4. タップ領域を設定
5. アクションを設定（URL、テキスト、クーポンなど）

#### 方法2: Messaging APIで設定
- **メリット**: コードで管理できる、複数パターンを簡単に切り替え可能
- **デメリット**: プログラミング知識が必要

**実装例：**
```javascript
// api/create-rich-menu.js（Serverless Function）
const line = require('@line/bot-sdk');

const client = new line.Client({
  channelAccessToken: process.env.CHANNEL_ACCESS_TOKEN
});

async function createRichMenu() {
  const richMenu = {
    size: { width: 2500, height: 1686 },
    selected: true,
    name: '2025年1月メニュー',
    chatBarText: 'メニュー',
    areas: [
      {
        bounds: { x: 0, y: 0, width: 833, height: 843 },
        action: { type: 'uri', uri: 'https://love-counter-theta.vercel.app/' }
      },
      {
        bounds: { x: 833, y: 0, width: 834, height: 843 },
        action: { type: 'message', text: '占い' }
      },
      {
        bounds: { x: 1667, y: 0, width: 833, height: 843 },
        action: { type: 'message', text: '紹介' }
      }
    ]
  };

  const richMenuId = await client.createRichMenu(richMenu);

  // 画像をアップロード（別途必要）
  // await client.setRichMenuImage(richMenuId, fs.readFileSync('menu.png'));

  // デフォルトメニューに設定
  await client.setDefaultRichMenu(richMenuId);

  return richMenuId;
}
```
