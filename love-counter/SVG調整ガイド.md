# SVG調整ガイド - リンゴボタンの構造

## 📐 SVGの基本座標系

```
viewBox="0 0 100 100"
↑ 左上が(0,0)、右下が(100,100)の座標空間

     0                    100
  0  ┌─────────────────────┐
     │                     │
     │    (50, 50)         │
     │       ●             │
     │                     │
100  └─────────────────────┘

x軸 → 右に行くほど大きい
y軸 → 下に行くほど大きい
```

---

## 🍎 リンゴボタンの全コード（現在使用中）

```html
<svg viewBox="0 0 100 100" xmlns="http://www.w3.org/2000/svg">
  <!-- 1. リンゴの葉っぱ -->
  <path d="M 48,15 Q 45,8 40,10 Q 35,12 38,17" fill="#8bc34a" />

  <!-- 2. リンゴの茎 -->
  <path d="M 48,15 L 50,25" stroke="#8d6e63" stroke-width="2" fill="none" />

  <!-- 3. リンゴの本体 -->
  <path class="apple-body" d="
    M 50,25
    C 30,25 20,35 20,50
    C 20,65 30,80 45,85
    L 45,88
    C 45,92 47,95 50,95
    C 53,95 55,92 55,88
    L 55,85
    C 70,80 80,65 80,50
    C 80,35 70,25 50,25 Z
  " fill="#f48fb1" />

  <!-- 4. ハイライト（光沢） -->
  <ellipse cx="38" cy="40" rx="8" ry="12" fill="#ffc1e3" opacity="0.6" />

  <!-- 5. テキスト -->
  <text class="button-text" x="50" y="58">YES</text>
</svg>
```

## 🔍 各パーツの詳細解説

### 1️⃣ 葉っぱ（path要素）

```html
<path d="M 48,15 Q 45,8 40,10 Q 35,12 38,17" fill="#8bc34a" />
```

**コマンドの意味：**
```
M 48,15    → Move: (48, 15)に移動（開始点）
Q 45,8 40,10 → Quadratic curve: 制御点(45,8)を通って(40,10)へ曲線
Q 35,12 38,17 → もう一度曲線で(38,17)へ
```

**調整ポイント：**
- `fill="#8bc34a"` → 葉っぱの色（緑）
- 数値を変えると形が変わる

---

### 2️⃣ 茎（path要素）

```html
<path d="M 48,15 L 50,25" stroke="#8d6e63" stroke-width="2" fill="none" />
```

**コマンドの意味：**
```
M 48,15    → (48, 15)に移動
L 50,25    → Line: (50, 25)まで直線
```

**調整ポイント：**
- `stroke="#8d6e63"` → 線の色（茶色）
- `stroke-width="2"` → 線の太さ（数字を大きくすると太くなる）
- `fill="none"` → 塗りつぶしなし

---

### 3️⃣ リンゴ本体（path要素）

```html
<path d="
  M 50,25
  C 30,25 20,35 20,50
  C 20,65 30,80 45,85
  L 45,88
  C 45,92 47,95 50,95
  C 53,95 55,92 55,88
  L 55,85
  C 70,80 80,65 80,50
  C 80,35 70,25 50,25 Z
" fill="#f48fb1" />
```

**コマンドの意味：**
```
M 50,25              → (50, 25)から開始（リンゴの上）
C 30,25 20,35 20,50  → Cubic curve: 左側のカーブ
C 20,65 30,80 45,85  → 左下のカーブ
L 45,88              → 下部のくぼみ（左）
C 45,92 47,95 50,95  → くぼみの底
C 53,95 55,92 55,88  → くぼみから上がる
L 55,85              → 右下へ
C 70,80 80,65 80,50  → 右側のカーブ
C 80,35 70,25 50,25  → 右上のカーブ
Z                    → 閉じる
```

**調整ポイント：**
- `fill="#f48fb1"` → リンゴの色（ピンク）
- 数値を変えるとリンゴの形が変わる

---

### 4️⃣ ハイライト（ellipse要素）

```html
<ellipse cx="38" cy="40" rx="8" ry="12" fill="#ffc1e3" opacity="0.6" />
```

**属性の意味：**
```
cx="38"          → 中心のx座標
cy="40"          → 中心のy座標
rx="8"           → 横の半径
ry="12"          → 縦の半径
fill="#ffc1e3"   → 色（明るいピンク）
opacity="0.6"    → 透明度（0=透明、1=不透明）
```

**調整ポイント：**
- `cx, cy` → ハイライトの位置
- `rx, ry` → ハイライトの大きさ
- `opacity` → 明るさ

---

### 5️⃣ テキスト（text要素）

```html
<text class="button-text" x="50" y="58">YES</text>
```

**属性の意味：**
```
x="50"    → テキストの中心x座標
y="58"    → テキストのベースラインy座標
```

**CSSで制御：**
```css
.button-text {
  font-size: 22px;        /* 文字サイズ */
  font-weight: bold;      /* 太字 */
  fill: white;            /* 文字色 */
  text-anchor: middle;    /* 中央揃え */
}
```

## 🔧 path要素のコマンド一覧

| コマンド | 意味 | 例 |
|---|---|---|
| **M x,y** | Move（移動） | `M 50,25` |
| **L x,y** | Line（直線） | `L 50,80` |
| **C x1,y1 x2,y2 x,y** | Cubic curve（3次曲線） | `C 30,25 20,35 20,50` |
| **Q x1,y1 x,y** | Quadratic curve（2次曲線） | `Q 45,8 40,10` |
| **Z** | Close path（閉じる） | `Z` |

---

## 🎨 Q（2次曲線）コマンドの仕組み - 詳しく解説

### 基本構文
```
Q x1,y1 x,y
  ↑    ↑
  制御点 終点
```

**3つの点で曲線を作る：**
1. **始点** = 前のコマンドの終了位置（Mで指定した位置など）
2. **制御点（x1, y1）** = 曲線を引っ張る方向を決める点
3. **終点（x, y）** = 曲線の終わり

### 実例：リンゴの軸

```html
<!-- 下向き曲線 -->
<path d="M 45,28 Q 50,33 55,28" />
```

**分解して理解：**
```
M 45,28      ← 始点：左端（45, 28）に移動
Q 50,33      ← 制御点：中央下（50, 33）に向かって引っ張る
  55,28      ← 終点：右端（55, 28）で終わる
```

### 視覚的に理解する

```
y=28  (45,28)●─────────●(55,28)  ← 始点と終点（同じ高さ）
          ＼         ／
           ＼       ／
            ＼     ／
             ＼   ／
              ＼ ／
           ●(50,33)  ← 制御点（下）

結果：下向きの曲線ができる 🍎
```

**制御点を上に変えると：**
```html
<path d="M 45,28 Q 50,23 55,28" />
                     ↑ y=23（上）
```

```
           ●(50,23)  ← 制御点（上）
              ／ ＼
             ／   ＼
            ／     ＼
           ／       ＼
          ／         ＼
y=28  (45,28)●─────────●(55,28)

結果：上向きの曲線ができる 🍎
```

### 制御点の位置で曲線の形が変わる

| 制御点のy座標 | 結果 | 理由 |
|---|---|---|
| y < 28（上） | 上向き曲線 | 始点・終点より上に引っ張られる |
| y > 28（下） | 下向き曲線 | 始点・終点より下に引っ張られる |
| y = 28（同じ） | 直線 | 引っ張られない |

### まとめ

**Q コマンド = ゴムひもを引っ張るイメージ**

```
始点 ●─────────● 終点
      ＼     ／
       ＼   ／     ← このゴムひもを...
        ● 制御点    ← ここで引っ張る！
```

- 制御点を**上**に置く → 上向き曲線
- 制御点を**下**に置く → 下向き曲線
- 制御点を**横**に置く → 横向き曲線

**数値の変更で形が変わる理由：**
→ **制御点の位置が変わる = ゴムを引っ張る方向が変わる！**

## 🎯 調整の手順

### 例：リンゴを左に移動したい

1. **リンゴ本体のx座標を全部-10する**
```html
<!-- 元 -->
M 50,25 C 30,25 20,35 20,50 ...

<!-- 左に10移動 -->
M 40,25 C 20,25 10,35 10,50 ...
```

2. **ハイライトも移動**
```html
<!-- 元 -->
<ellipse cx="38" cy="40" ... />

<!-- 左に10移動 -->
<ellipse cx="28" cy="40" ... />
```
