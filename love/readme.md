# さみしくなったら押してねボタン（健全版 & えろかわ版）

## 🎯 コンセプト

- 「〇〇だったらこのボタン押してね？」という誘導文を表示し、
- ボタンを押すとランダムで「送ってほしいメッセージ」が表示されるミニWebアプリ。
- モード切り替えで  
  - 🍀 健全（safe）  
  - 🔥 えろかわ（ero）  
  を使い分け可能。

スマホで開く前提の、1ページ完結のシンプル構成。

---

## 🧱 技術スタック

- HTML / CSS / JavaScript（プレーンでOK）
- ビルドなし。`index.html` + 必要なら `script.js` 程度。

---

## 🌈 画面仕様

### 画面の構成

- モード切り替えボタン（safe / ero）
- トリガーテキスト（「〇〇だったらこのボタン押してね？」をランダム表示）
- メインボタン（ぽちっ 💗）
- メッセージ表示
- コピーボタン
- コピー状態テキスト
- 押された回数表示（モード別 or 共通は実装者に任せる）

### 推奨DOM構造（ID / class）

- モード切り替え：  
  - `id="modeSafe"`  
  - `id="modeEro"`
- トリガーテキスト： `id="triggerText"`
- メインボタン： `id="mainButton"`
- メッセージ： `id="message"`
- コピーボタン： `id="copyButton"`
- コピー状態： `id="copyStatus"`
- カウンター： `id="count"`
- カード： `class="card"`
- body にモード用クラス：  
  - `safe-mode`  
  - `ero-mode`

---

## 🔀 モード仕様（safe / ero）

### 管理方法

- `currentMode` を `"safe"` / `"ero"` として保持。
- モード切り替え時：
  - `currentMode` 更新
  - スタイル変更（背景色など）
  - トリガーテキストをそのモードのものからランダム選択
  - カウント表示も必要なら切り替え

### 永続化（任意）

- localStorage に `lastMode` を保存し、次回ロード時に復元。
